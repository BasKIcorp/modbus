# ModBus REST API

## Общий вид запросов
### Запрос на чтение одного регистра (номер функции - 3)
- GET-запрос: /lab_num/device_num/get_function

### Запрос на запись (номер функции - 16), n - записываемое значение
- POST-запрос: /lab_num/device_num/set_function?value=n
